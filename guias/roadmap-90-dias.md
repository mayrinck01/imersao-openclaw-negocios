# Roadmap de 90 Dias — Template

> Plano de implementação pós-imersão. Saia da imersão com isso preenchido.

---

## Semana 1 (Dias 1-7): Consolidar o que fez na imersão

- [ ] Revisar e completar `empresa/contexto/empresa.md` (dados que faltaram)
- [ ] Preencher contexto de pelo menos 2 áreas
- [ ] Garantir que o cron criado na imersão está rodando sem erros
- [ ] Adicionar mais 1-2 skills baseadas nas necessidades do dia a dia
- [ ] Configurar heartbeat (1h) para monitorar saúde dos crons
- [ ] Configurar sync diário para GitHub (cron 21h)

**Meta da semana:** agente respondendo perguntas sobre a empresa com contexto real.

---

## Semanas 2-3 (Dias 8-21): Expandir para todas as áreas

- [ ] Preencher contexto de TODAS as áreas relevantes
- [ ] Conectar mais fontes de dados (planilhas, APIs)
- [ ] Criar 1 skill por área (mínimo)
- [ ] Documentar pelo menos 3 rotinas
- [ ] Configurar crons para as rotinas mais importantes
- [ ] Registrar primeiras decisões em `empresa/decisoes/`
- [ ] Registrar primeiras lições em `empresa/gestao/licoes.md`

**Meta da semana 3:** cada área com contexto + pelo menos 1 skill + 1 rotina.

---

## Mês 1 (Dias 22-30): Equipe usando o agente

- [ ] Criar grupo/tópicos no Telegram para a equipe
- [ ] Configurar mapeamento área ↔ tópico (`memory/channels.md`)
- [ ] Onboardar equipe: ensinar a falar com o agente
- [ ] Definir permissionamento (quem acessa o quê)
- [ ] Configurar `allowFrom` e modo ask para ações sensíveis
- [ ] Primeiro relatório semanal consolidado (cross-área)

**Meta do mês 1:** equipe inteira usando o agente no dia a dia.

---

## Mês 2 (Dias 31-60): Otimizar e automatizar

- [ ] Revisar skills existentes: quais estão sendo usadas? Quais não?
- [ ] Criar skills mais avançadas (que combinam dados de múltiplas fontes)
- [ ] Avaliar se precisa de agentes por área (gargalo de concorrência?)
- [ ] Se sim: criar 1º agente especializado (área mais movimentada)
- [ ] Consolidar base de conhecimento: FAQ, processos, playbooks
- [ ] Revisar e atualizar métricas em `empresa/contexto/metricas.md`
- [ ] Documentar lições e decisões do mês

**Meta do mês 2:** agente é parte da rotina da empresa, não novidade.

---

## Mês 3 (Dias 61-90): Escalar

- [ ] Avaliar ROI: quanto tempo o agente economiza por semana?
- [ ] Criar mais agentes especializados (se necessário)
- [ ] Conectar ferramentas mais complexas (CRM, plataforma de vendas, etc.)
- [ ] Treinar agente com histórico de decisões (decisões passadas informam futuras)
- [ ] Implementar use case cross-área (relatório consolidado semanal)
- [ ] Criar sub-agentes para tarefas específicas (se escala justificar)
- [ ] Avaliar: o que mais pode ser automatizado?

**Meta do mês 3:** empresa rodando com agente(s) como parte integral da operação.

---

## Métricas de Sucesso

Ao final dos 90 dias, avaliar:

| Métrica | Alvo |
|---------|------|
| Áreas com contexto completo | 100% |
| Skills ativas | 5+ |
| Crons rodando sem erro | 3+ |
| Equipe usando o agente | 80%+ |
| Tempo economizado/semana | 5h+ |
| Decisões registradas no repo | 10+ |
| Relatórios automáticos | Semanal |
