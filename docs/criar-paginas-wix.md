# Criar Páginas Termos e Privacidade no Wix

## Por que não funciona via API

A API REST do Wix **não permite criar novas páginas** diretamente.
Para criar páginas, precisa ser feito manualmente pelo Wix Editor.

## PASSO A PASSO MANUAL (5 minutos)

### 1. Acesse o Wix Editor
- Vai em: https://www.wix.com
- Clica em "Gerenciar Site" → "Editar Site"

### 2. Criar Página Termos de Serviço
1. No menu lateral esquerdo → **Páginas**
2. Clica em **"+ Adicionar Página"**
3. Nome: **Termos de Serviço**
4. Tipo: Página em branco
5. Slug: `termos-de-servico`
6. Clica em **Salvar**

### 3. Criar Página Privacidade
1. Mesmo processo
2. Nome: **Política de Privacidade**
3. Slug: `politica-de-privacidade`
4. Clica em **Salvar**

### 4. URLs finais
- https://www.cakeco.com.br/termos-de-servico
- https://www.cakeco.com.br/politica-de-privacidade
