11h35 Bloco 6 — Segurança 15 min
FALA — BLOCO CONTÍNUO
Segurança — o elefante na sala
"E se o agente fizer besteira?" — Pergunta justa.
3 camadas de proteção:
Camada 1 — Servidor: a fundação
SSH key-only, Fail2ban, UFW Firewall, Gateway localhost, Updates automáticos
90% das empresas não faz o básico. Já bloqueia Prompt Injection e Privilege Abuse.
Camada 2 — Agente: o comportamento
dmPolicy allowlist, .env chmod 600, skills auditadas, tool restrictions, requireMention
Analogia: crachá do estagiário — entra no prédio, mas só nos andares que você definiu
Camada 3 — Processo: disciplina operacional
Dupla autorização, audit crons, logs com timestamp, rotação de tokens 90 dias, memória no GitHub
